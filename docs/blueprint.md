# **App Name**: StockMaster

## Core Features:

- Product Stock Management: Allows adding, modifying, and disabling products. Disabling a product keeps it in the database for historical/forensic reasons, but prevents it from being actively managed or displayed.
- Stock Level Tracking: Provides real-time tracking of product stock levels.
- Sales Analysis Dashboard: Presents data on best-selling products through charts and graphs, giving insights into sales trends.
- User Authentication: Secure login system with admin-level privileges for designated personnel.
- Sales Forecasting Tool: Uses historical sales data to predict future demand for products using machine learning algorithms. Considers seasonal trends and other relevant factors to provide estimates that the staff will review using the LLM as a tool.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#ECEFF1), almost white, to ensure comfortable readability and a clean aesthetic.
- Accent color: Yellow-orange (#FFAB40), approximately 30 degrees 'left' of blue on the color wheel, to highlight key interactive elements and important information. Use to create contrast and call attention to actions like saving changes or critical alerts.
- Body and headline font: 'Inter' sans-serif for a clean and modern look.
- Simple and intuitive icons for product management and data visualization.
- Clean, organized layout with a focus on usability for backend users.
- Subtle transitions and animations to provide feedback on user actions.
